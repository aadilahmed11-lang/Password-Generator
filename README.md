This is a simple Python project made by a student. The program generates a random password based on user input.
WHAT THIS PROGRAM DOES:
      - Takes password length from the user
      - Asks if numbers should be included
      - Asks if special characters should be included
      - Generates a random password
      - Option to save the password in a text file
FILES USED:
      - proj 7.py : Main Python file
      - passwords.txt : Stores saved passwords
HOW TO RUN:
      1. Open terminal
      2. Go to the file location
      3. Run: python proj 7.py
CONCEPTS:
      - Functions
      - Loops
      - If-else
      - User input
      - File handling
